Mobile Application for Classification of Corn Disease Using Image Processing

This is our capstone project, completed as a team of three. I played a key role as the web developer, responsible for building a website using native PHP, HTML/CSS, and Bootstrap that functioned as the central database for the system. All data collected from the Android mobile application was stored on the website, and I handled the communication between the website and the Android app, ensuring seamless data requests and transfers.

I developed a data map for both the website and the Android application to visually display the distribution of corn diseases across different areas. In addition to my web development tasks, I contributed to the Android application by developing the image processing model using Python, achieving a 92% accuracy in classifying corn diseases.

The website featured a report generation tool that allowed users to export data into Excel files for easy downloading and analysis. I hosted the website on my own machine using XAMPP with port forwarding, enabling access without external hosting costs. The entire project was completed at zero cost, showcasing our ability to deliver a functional, cost-effective solution.

Llanera Teachers Association Multi-Purpose Cooperative

The Llanera Teachers Association Multi-Purpose Cooperative (LTAMPC) is a small association that offers services such as loans, savings, and other financial transactions. During my OJT as a full-stack developer, I noticed their manual process for managing member records and transactions was time-consuming.

To streamline their workflow, I developed a web-based profiling system using native PHP, HTML/CSS, and Bootstrap. This system allowed the staff to quickly search for and manage member information, saving significant time compared to their manual processes. I also implemented an automatic age calculation feature to ensure member ages were always current.

Additionally, I created a function that allowed the cooperative to generate filtered search results into Excel files, making data extraction more efficient. I set up the system using XAMPP, enabling all computers on their local network to input or delete data from the database in real time. This solution improved the cooperative’s daily operations, providing a cost-effective and user-friendly system.

San Jose DepEd Region III Survey Website

During my time assisting at San Jose DepEd Region III, the OJTs reached out for my help with their website development. They needed support with back-end functionality, as they had limited experience in that area. I also contributed to improving the website’s design to enhance user-friendliness.

Using native PHP, Bootstrap, HTML/CSS, XAMPP, Font Awesome, and various necessary plugins, I developed a website that functions similarly to Google Forms. This setup allowed them to handle sensitive survey data internally, ensuring secure management of information exchange.

The website features an admin panel where administrators can manage all data submitted by users. The admin page also includes a function to generate filtered survey reports as PDF files, facilitating efficient data analysis and reporting.

Additionally, I implemented a function to read Excel files, enabling the import of data directly into the MySQL database. This streamlined the onboarding process for users and improved overall efficiency.

My contributions helped the team establish a functional and secure online platform tailored to their needs.

Movie Review Website

During my time at university, I was required to build a fully functioning website using Laravel, and I was assigned a movie review project. The website features user login credentials, allowing users to select a movie and rate it.

My primary task was to develop a system that collects all reviews for a specific movie, calculates the average rating, and displays it alongside all user comments. I achieved this by implementing a robust back-end logic that aggregates the data effectively, ensuring that users could see both the overall reception of a movie and individual feedback from other viewers.

The project not only enhanced my skills in Laravel but also gave me valuable experience in building user-centric applications that handle data collection and display efficiently.
